# 目的 #

让本规则尽可能全面的支持主流的广告过滤软件，使用户无需费力更换广告过滤软件，只要使用当前配置即可达到最佳效果。

# 局限性 #

已知确实无法支持的广告过滤软件：
**傲游浏览器**
